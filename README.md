# Gestao de Eventos

## React Native app

```
npx expo whoami
```

if you have a account in expo.dev
```
npx expo login
```
if you don't have account
```
npx expo register
```
later run
```
npm install
```

```
npm run dev
```

### Install expo on your device
### Scan the QR code with your device or use  Android Studio in your computer
